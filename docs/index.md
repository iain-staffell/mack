# The MAC Connector Reference

This section contains connector-specific documentation for the APIs currently implemented in MAC.

Each connector page focuses on how MAC uses that source in practice:

- what the connector is for
- which upstream endpoint it calls
- authentication requirements
- parameters accepted by MAC
- defaults applied by MAC
- the output shape returned by MAC
- known limits, licensing notes, and caveats
