# The MACK Connector Reference

This section contains connector-specific documentation for the APIs currently implemented in MACK.

Each connector page focuses on how MACK uses that source in practice:

- what the connector is for
- which upstream endpoint it calls
- authentication requirements
- parameters accepted by MACK
- defaults applied by MACK
- the output shape returned by MACK
- known limits, licensing notes, and caveats
